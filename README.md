# photogram

upload images, manage their posts such assaving to favorites and taging relevant posts

HIGHLIGHTS:
-user authentication

-share on facebook

-using redux for global state management

-responsive web design

TECHNOLOGIES

-Front: React-redux

-Style: Scss

-Backend: Node.js, Express

-DB: MongoDB

-Image host: Cloudinary

-App host: Heroku
